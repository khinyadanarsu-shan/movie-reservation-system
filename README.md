# movie-reservation-system
A web-based movie ticket booking system with seat selection, scheduling and booking management using PHP, MySQL, Html, CSS and JavaScript.

Features
User registration and login
Movie listing and details
Seat reservation system
Booking confirmation
Email notifications using PHPMailer
Admin management panel
Technologies Used
PHP
MySQL
HTML
CSS
JavaScript
PHPMailer
Installation
Clone this repository:
Move the project folder to the XAMPP htdocs directory.
Import the database.sql file into MySQL.
Update the database configuration in config.php.
Start Apache and MySQL in XAMPP.
Open the project in your browser:
http://localhost/movie-reservation-system
